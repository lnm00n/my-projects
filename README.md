# my-projects
Overview

This repository contains my personal developer portfolio, built to showcase my projects, technical skills, and experience in development and QA automation.

The portfolio is built using a component-based architecture in Vue.js, allowing for modular, reusable UI components and easier scalability.

Supabase is used to handle backend functionality, including storing and retrieving data for the contact form. When a user submits the form, the data is sent to the database, enabling persistent storage and structured data management.

Playwright is used to automate key user flows and ensure the reliability of the user interface through end-to-end testing.

Tech Stack
Frontend: Vue.js, HTML, CSS
Backend / Database: Supabase
Testing: Playwright
Build Tool: Vite

Getting Started
git clone https://github.com/lnm00n/portfolio.git
cd my-projects
npm install
npm run dev
